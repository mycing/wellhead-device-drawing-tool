# 井口裝置繪圖工具

一款用於繪製井口裝置示意圖的 Windows 桌面應用程序。

![.NET Framework](https://img.shields.io/badge/.NET%20Framework-4.7.2-blue)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)
![License](https://img.shields.io/badge/License-MIT-green)

## 功能特點

- **豐富的內置裝置庫** - 包含萬能防噴器、閘板防噴器、套管頭、轉盤面等 20+ 種常用井口裝置
- **自定義裝置** - 支持導入自己的裝置圖片（JPG、PNG、SVG 等格式）
- **靈活的標籤系統** - 樹狀結構管理文字標籤，支持多層級分類
- **模板庫** - 保存常用的裝置組合為模板，一鍵載入重複使用
- **智能對齊** - 自動對齊功能快速整理畫布上的裝置和文字
- **便捷截圖** - 支持手動截圖、自動截圖和長圖拼接
- **上色/線條切換** - 部分裝置支持彩色填充和線條輪廓兩種樣式

## 界面預覽

軟件界面分為四個主要區域：
- **左側** - 井口裝置選擇列表
- **中間上方** - 標籤管理樹
- **中間下方** - 已保存的模板庫
- **右側** - 繪圖畫布

## 快速開始

1. **放置裝置** - 在左側列表選擇裝置，點擊畫布放置
2. **添加標籤** - 在標籤樹選擇文字，點擊畫布放置
3. **調整佈局** - 拖動移動位置，滾輪縮放大小
4. **自動對齊** - 右鍵菜單「自動對齊」一鍵整理
5. **保存/截圖** - 保存為模板或截圖到剪貼簿

## 操作說明

### 畫布操作
| 操作 | 說明 |
|------|------|
| 左鍵點擊 | 放置選中的裝置或標籤 |
| 左鍵拖動 | 移動裝置或標籤位置 |
| 滾輪（在控件上） | 縮放裝置或標籤大小 |
| 滾輪（在空白處） | 上下滾動畫布 |
| 右鍵點擊 | 打開功能菜單 |

### 右鍵菜單功能
- **刪除** - 刪除選中的控件
- **自動對齊** - 自動排列所有控件
- **自動截圖** - 智能識別內容範圍並截圖
- **截圖** - 手動選擇區域截圖
- **清空畫布** - 清除所有內容
- **添加樣例到庫** - 保存當前畫布為模板

## 系統要求

- Windows 7 / 8 / 10 / 11
- .NET Framework 4.7.2 或更高版本

## 安裝使用

1. 從 [Releases](https://github.com/mycing/wellhead-device-drawing-tool/releases) 下載最新版本
2. 解壓到任意目錄
3. 運行 `4.18.exe`

## 數據存儲

程序數據保存在軟件所在目錄：
- `template_library.bin` - 模板庫
- `tagtree_items.bin` - 標籤樹
- `pictures/` - 自定義裝置圖片

> 備份時請複製整個程序資料夾
